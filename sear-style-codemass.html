ItheereumCybernetics

This module encapsulates all the visual DNA of the 'Sear' app. It defines the color philosophy, the "orthogonally correct" animations, and the unique shape interactions developed in an iterative process. The base of the style is a dark palette that uses "Reflexive Onyx" and "Dark Galaxy Blue", accentuated by bright "dark light purple" and "crystalline bright turquoise" tones. These are defined as CSS root variables to ensure consistency and easy maintainability. Itheereum Cybernetics, Berlin (EU) 2025.
